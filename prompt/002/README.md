# 提示词生成器

一个纯原生JavaScript、HTML和CSS实现的提示词生成工具，可以帮助用户创建精准、生动的提示词，用于AI图像生成等场景。

## 功能特点

- 基础提示词输入
- 丰富的选项选择：人物姿势、表情、场景、景别、风格等
- 特殊选项：仅参考本次图、重置风格关联、去除元素等
- 一键复制生成的提示词
- 清空所有输入功能
- 响应式设计，适配不同屏幕尺寸
- 支持深色模式

## 技术栈

- HTML
- CSS
- JavaScript（纯原生，无依赖）

## 使用方法

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件即可使用
3. 在GitHub Pages上部署时，确保正确设置路径

## 部署说明

本项目可以直接部署到任何静态网站托管服务，如GitHub Pages、Netlify、Vercel等。只需上传所有文件即可。

## 开发说明

如需修改样式，可以编辑 `styles.css` 文件。
如需修改功能逻辑，可以编辑 `script.js` 文件。
如需修改页面结构，可以编辑 `index.html` 文件。
