# 提示词生成器

一个简单易用的提示词生成工具，用于创建精准、生动的提示词描述。

## 特点

- 无需安装任何依赖，直接双击 `index.html` 即可使用
- 支持多种提示词选项，包括姿势、表情、场景、景别、风格等
- 响应式设计，适配各种屏幕尺寸
- 支持复制生成的提示词到剪贴板
- 支持多种特殊指令，如"仅参考本次图"、"重置风格关联"等

## 使用方法

1. 直接双击 `index.html` 文件打开应用
2. 在"基础提示词"输入框中输入您的基本描述
3. 选择您需要的各种选项（姿势、表情、场景等）
4. 系统会自动生成完整的提示词
5. 点击"复制提示词"按钮即可将生成的提示词复制到剪贴板

## 功能说明

- **基础提示词**：输入您的核心描述
- **人物姿势**：选择30种不同的人物姿势
- **人物表情**：选择30种不同的人物表情
- **场景**：选择各种不同的场景背景
- **景别**：选择不同的拍摄景别
- **风格**：选择不同的艺术风格
- **瞎呲程度**：可分别设置环境、皮肤、衣服的瞎呲程度
- **特殊指令**：
  - 仅参考本次图：完全以提供的参考图为视觉基准
  - 重置风格关联：重置风格记忆，仅聚焦当前参考图
  - 去除元素：去除文字、英文、图标、水印、LOGO

## 注意事项

- 本工具仅用于生成提示词，不包含实际的图像生成功能
- 生成的提示词可以用于各种AI图像生成工具
- 如需保存生成的提示词，请手动复制并保存到其他位置
