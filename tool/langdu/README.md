# 早读助手 - 项目运行指南

## 项目介绍
这是一个用于记录和分析学生早读声音的前端应用，支持录音、音量监测和录音时长统计功能。

## 环境要求
- Node.js (推荐 v16.0 或更高版本)
- pnpm (项目使用的包管理工具)

## 快速开始

### 步骤 1: 安装依赖
打开命令行工具，进入项目目录，执行以下命令安装依赖：

```bash
pnpm install
```

### 步骤 2: 启动开发服务器
依赖安装完成后，执行以下命令启动本地开发服务器：

```bash
pnpm dev
```

服务器启动后，会自动打开浏览器并访问 `http://localhost:3000`。

### 步骤 3: 构建生产版本
如果需要构建生产版本的代码，执行以下命令：

```bash
pnpm build
```

构建后的文件会生成在 `dist` 目录中。

## 常见问题排查

### 1. 无法安装依赖
- 确保已安装 Node.js 和 pnpm
- 可以尝试使用 npm 或 yarn 替代：
  ```bash
  npm install
  # 或
  yarn install
  ```

### 2. 启动服务器失败
- 检查端口 3000 是否被占用
- 查看命令行输出的错误信息
- 确认依赖安装是否成功

### 3. 麦克风权限问题
- 首次使用时，浏览器会请求麦克风权限，请选择"允许"
- 如果之前拒绝了权限，需要在浏览器设置中重新授予权限

### 4. 其他问题
如果遇到其他问题，请检查浏览器控制台的错误信息，或联系技术支持。
